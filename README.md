# 26lab.pl

**Opis**  
Moja osobista strona, projekt stworzony do nauki w HTML, CSS i JavaScript.

**Technologie**  
- HTML5
- CSS
- Tailwind CSS (Tylko w about.html) 
- JavaScript (ES6+)
- AOS - scroll animation library

**Live Demo**  
https://26lab.pl

**Co udało mi się dzięki temu nauczyć**  
- Budowa responsywnego layoutu
- Organizacja struktury plików (HTML, CSS, JS)  
- Podstawy interakcji w JS  
- Samodzielne przeprowadzenie projektu
- Integracja strony z Google Analytics
  
> W projekcie występują drobne błędy z adaptacją na urządzeniach moblinych w navmenu oraz w sekcji oferta

**Jak uruchomić lokalnie**  
1. Sklonuj repozytorium:  
```
git clone https://github.com/<uzytkownik>/26lab.git
```  
2. Otwórz plik `index.html` w przeglądarce
